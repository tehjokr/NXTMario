NumBox
 

This RIC element can be used for displaying of numeric values on the NXT screen.

0 - 4095

 X X

 
 
Size of element: 12 Byte
   
Hint:

With the mouse you can define the position of output on the NXT screen by pressing the mouse button on the working area. Horizontal positioning is possible without any limitation – but the vertical position will be aligned to one of the 8 possible textual lines on the screen. When you define the position with the mouse than the attributes for “Pos X” and “Pos Y” will be set as a constant value.
