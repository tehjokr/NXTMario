RIC elements

 

Information

 
 

This RIC element is only required for NXT-G because it described the size of the RIC file. When you select a RIC file within NXT-G, than the program read the information about the size of the picture from this RIC element.

For displaying a RIC file on the NXT screen this element will be ignored.
