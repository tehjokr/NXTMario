RIC elements

 

Point
 

 The RIC element Point will be shown immediately on the NXT screen. When the coordinate of the point will be outside the visible area of the NXT screen than the point will not be drawn. Unfortunately you only can set a point – cleaning of single points on the NXT screen isn’t possible by using this RIC element.
