About nxtRICeditV2

 

I was in the happy position to take my hands on the brand new Lego Mindstorms NXT system - 6 months before it was official launched for the American market. And I detected, after building first programs with language NXT-G, that the total count of RIC files, which will be delivered with the NXT system, are limited. And there was no tool available for modifying or creating the RIC files.

 

This was the main reason for building the first version of nxtRICedit. Without the detail knowledge about the structure of the RIC files - and with a lot of Try-and-Error - I was able to reassemble the file format and created the first version of the program. Since eastern 2006 the tool is the world standard tool for modifying and creating of simple RIC files.

 

After official release of the source code for the new NXT system, I detected that the structure of RIC files is more complex with additional features, which was not known before.

 

Due to some private and job related reasons, unfortunately I was not able to complete and release the new version of nxtRICeditV2. My estimation is that more than 90% of all users, who would like to create or modify RIC files, will furthermore be happy with the first version of nxtRICedit. But the new version was developed for the other 10 % of users, who would like to get more out from the NXT ... Especially the programming language NBC/NXC which was developed by John Hansen, can completely use the new features of nxtRICeditV2.
