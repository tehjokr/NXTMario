RIC-Elemente

 

Line
 

Mit dem RIC-Element Line wird eine Linie direkt auf dem NXT-Bildschirm ausgegeben. Liegt der Start- oder Endpunkt der Linie außerhalb des darstellbaren Bereichs, so wird nur der noch sichtbare Teil der Linie auf dem NXT-Bildschirm ausgegeben.

Eingabetips: Mit der Maus kann sowohl der Startpunkt (Mausetaste drücken), als auch der Endpunkt (Maustaste loslassen) festgelegt werden. Die Linie wird bei gedrückter Maustaste ständig angezeigt. Eine Definition der Maus setzt alle vier Parameter für Start- und Endposition auf einen konstanten Wert zurück.
