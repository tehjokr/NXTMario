RIC-Elemente

 

Point
 

Das RIC-Element Point wird direkt auf dem Bildschirm ausgegeben. Wenn die Koordinate des Punktes außerhalb des darstellbaren Bereichs auf dem NXT-Bildschirms ist, dann wird der Punkt nicht mehr angezeigt. Leider kann nur ein einzelner dunkler Punkt damit gesetzt werden - das Löschen von einzelnen Punkten ist mit diesem RIC-Element nicht möglich.
