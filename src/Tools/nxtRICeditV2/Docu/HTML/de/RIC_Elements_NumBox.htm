RIC-Elemente

 

NumBox
 

Dieses RIC-Element kann für die Ausgabe von numerischen Werten auf dem NXT-Display benutzt werden.

 
 

Parameter

Funktion

Wertebereich

 Parameter

VarMap

Copy Opt

Wird von der aktuellen Firmware ignoriert - Parameter muss nicht verwendet werden

Vertikale Position der Ausgabe

0 - 4095

 X X

Value

Wert der angezeigt werden soll

0 - 4095

 X X

 
 
Elementgröße: 12 Byte
   
Eingabetips: Mit der Maus kann die Position der Ausgabe auf dem NXT-Bildschirm festgelegt werden. Dabei ist eine freie horizontale Position möglich. Vertikal rastet die Position nur bei den gültigen virtuellen Zeilen 1 bis 8 ein. Wird die Position mit der Maus eingegeben, so werden die Werte für die Position als konstanter Wert definiert.
