RIC-Elemente

 

Rectangle
 

Mit dem RIC-Element Rectangle wird ein Rechteck direkt auf dem NXT-Bildschirm ausgegeben. Liegt der Startpunkt innerhalb des NXT-Bildschirms, überschreitet jedoch die Breite und/oder Höhe den gültigen Bereich, dann wird sich der NXT beim Aufruf des RIC-Files aufhängen! Achten Sie bitte immer darauf, dass das Rechteck innerhalb des sichtbaren Bereichs liegt.

Eingabetips:

Mit der Maus können die beiden gegenüberliegende Ecken direkt eingegeben werden. Sobald die Maustaste gedrückt wird, wird die Erste festgelegt. Wenn die Maustaste wieder losgelassen wird, dann wird die zweite Ecke bestimmt. Eine Definition mit der Maus setzt die vier Werte Pos X, PosY, Width und Height als konstanten Wert.
